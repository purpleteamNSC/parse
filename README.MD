
## Variáveis de Ambiente

Para rodar esse projeto, você vai precisar instalar :

`Python 3 `

`Syslog-ng`


## Rodando localmente

Atualize o linux

```bash
  sudo apt-get update
```

Instale o python

```bash
  sudo apt-get install python3 -y
```

Instale o syslog

```bash
  sudo apt-install syslog-ng -y
```
